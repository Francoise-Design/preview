How to install manually in Figma

  1. Download and extract the .zip archive to a permanent folder on your computer.
  2. Open the Figma Desktop App (manual installation is not supported in the browser).
  3. In the menu, go to Plugins > Development > Import plugin from manifest...
  4. Select the manifest.json file located inside your extracted folder.
  5. Françoise Check will now be available in your development plugins list, ready to run locally.
